## Understand
Problem: Difficult to instantly deliver a parcel (central london)

Objective: To create an instant, paperless delivery service.

## MVP
CUSTOMER:
  - create a delivery
    - name
    - pick-up address
    - drop-off address

COURIER:
- list all possible delivery requests
  - select a delivery
  - confirm selection
